Key Concepts Summary
Functional Interface

Rules:

    1 abstract method
    any number of default methods
    any number of static methods

Example

    Runnable
    Callable
    Comparator
    Predicate

Lambda Expression Benefits:
    Less code
    No anonymous class
    Functional programming support
    Better readability
    Works well with Streams API

Real Industry Usage

Lambda expressions are heavily used in:

    Spring Framework
    Hibernate
    Apache Kafka

especially with:

    Streams
    Collections
    Async programming
    Event handling
